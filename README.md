# Extensiones ITSC – versión compacta

Sitio estático para consulta de extensiones del ITSC.

## Estructura
- `index.html` – aplicación
- `assets/logo.png` – logo institucional (reemplázalo por el oficial si deseas)

## Publicar en GitHub Pages
1. Crea un repositorio público (por ejemplo, `extensiones-itsc`).
2. Sube **los archivos descomprimidos** (no subas el ZIP).
3. Ve a **Settings → Pages**.
4. En **Build and deployment → Source**, selecciona **Deploy from a branch**.
5. Elige **Branch: main** y **/ (root)**. Guarda.
6. Espera el deploy y abre la URL que te muestra GitHub Pages.

> Nota: `.nojekyll` evita procesado de Jekyll y que la carpeta `assets/` funcione sin problemas.
